## 5. 最小可推广版本 (MVP Checklist)

以下 **5 项核心任务** 是对外推广的最低要求：

- [ ] **README 优化**：一句话说明 + 效果截图 + 快速开始命令
- [ ] **示例数据包**：打包 10 张测试图片 + config.yaml
- [ ] **一键安装**：`pip install` 或 `curl | bash` 安装脚本
- [ ] **效果 GIF**：30 秒内展示从照片到 3D 模型的过程
- [ ] **性能基准**：100 张照片 / 高斯模型 / 所需时间 / 显存
